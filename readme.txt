
STOPWATCH
-it have functionality like statrt,pause,stop and laps.
-it store the last 10 timer.
-it store 10 laps.
-it sync time if browser get closed or referesh.


Start/pause button : Same button to start and pause stopwatch.
Stop button :it stop the stopwatch and show the stopped time.
Lap Button : it will create laps and shows on screen.
Reset Button : it reset stopwatch and if laps is created then it will store into history.


Requirement:
-Chrome Browser.
-JavaScript enabled on browser.

How to use:

- Copy the Folder and make sure you have JS and CSS folder inside that folder.
- Open the index.html file.
- If default browser is not chrome then right click on index.html and then select
  open with and then select chrome browser.
